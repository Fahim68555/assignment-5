Question-3:What is event delegation in the DOM? in the DOM?
Event delegation is a technique in JavaScript where instead of attaching event listeners directly to individual elements, you attach a single event listener to a parent element. This event listener listens for events on its child elements, allowing you to handle events for dynamically added elements or many child elements efficiently.
Question-4: What is event bubbling in the DOM?
Event bubbling is a concept in the DOM (Document Object Model) that describes the way events propagate through the DOM tree when they are triggered on an element.
Question-5: How do you create, add, and remove elements using JavaScript?
Create: Use document.createElement() to create a new element.
Add: Use methods like appendChild() or insertBefore() to add the element to the DOM.
Remove: Use removeChild() or remove() to remove an element from the DOM.
